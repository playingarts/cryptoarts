# Crypto Arts

## Running locally

Make sure you have created `.env` file in the root of the project and set the NODE_ENV:

```
NODE_ENV=development
```

To run the app and watch for changes, use:

```
nodemon
```

If you want to run the app without the watch, use `yarn dev`.

Default `PORT` is `3000`.

Storybook is available with the `yarn storybook` command.
